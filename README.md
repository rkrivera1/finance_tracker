# Nextcloud Finance Tracker

## Overview
A comprehensive personal finance tracking application integrated with Nextcloud.

## Features
- Account Management
- Budget Tracking
- Transaction Recording
- Investment Monitoring

## Installation
1. Clone this repository into your Nextcloud `custom_apps` directory
2. Enable the app through Nextcloud's app management interface
3. Start tracking your finances!

## Requirements
- Nextcloud 24+
- PHP 8.1+

## Development
- Clone the repository
- Run `composer install` to install dependencies
- Develop and test locally
